# Task Assignment App

A Task Assignment app made with React and Redux.

## Install

Install and start the app:

```shell
  git clone https://github.com/pjmanikanda-ui/svaantech.git
  cd svaantech
  npm install
  npm start
```

The app will be available at: <http://localhost:3000/>
